OpenSARShip

OpenSARShip: Sentinel-1 ship interpretation

An open-access, large-scale dataset combining Sentinel-1 SAR imagery with Automatic Identification System (AIS) ship information. It has been widely used as a benchmark for developing and evaluating ship interpretation methods and supporting coastal monitoring applications.
