Publish this folder with GitHub Pages.

Recommended steps:
1. Create a GitHub repository named lanqinghuang.github.io, or another name you prefer.
2. Upload the CONTENTS of this publish folder, not the parent PersonalWeb folder.
3. In the GitHub repository, go to Settings > Pages.
4. Choose "Deploy from a branch".
5. Select branch "main" and folder "/root".
6. After GitHub finishes deployment, open the Pages URL shown in Settings > Pages.

This publish folder intentionally excludes private/source files such as CV_Huang.pdf,
Research_statement.docx, reply2key.docx, local PDFs, HEIC originals, unused code/news pages,
and intermediate image files.
